
# Educação Positiva — Entrega Final

Projeto final da disciplina: plataforma web para ONG fictícia Educação Positiva.
Inclui HTML, CSS, JavaScript, acessibilidade e instruções para deploy no GitHub Pages.

## Como testar localmente
1. Descompacte a pasta.
2. Abra o terminal dentro da pasta do projeto.
3. Rode `python -m http.server 8000` e abra `http://localhost:8000` no navegador.

## Como publicar no GitHub Pages
1. Crie um repositório público no GitHub (ex.: EducacaoPositiva).
2. Faça push dos arquivos para a branch `main`.
3. Vá em Settings → Pages e selecione `main` / root.
4. Aguarde alguns minutos e acesse `https://SEU_USUARIO.github.io/EducacaoPositiva/`.
